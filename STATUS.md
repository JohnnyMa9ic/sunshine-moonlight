# STATUS

Append one line per event using:

`YYYY-MM-DD HH:MM [side] message`

Examples:

`2026-04-18 10:12 [victus] Sunshine restarted, debug logging enabled, host idle`
`2026-04-18 10:14 [mac] Moonlight retry started at 1080p60 H.264 SDR`
`2026-04-18 10:15 [victus] Captured launch-window Sunshine log and committed to logs/`
`2026-04-18 11:26 [victus] Clean stable Sunshine reinstall completed; ultrawide target {f6352b4e-ca82-5c2f-8077-47cd6444caff} applied in web UI; Moonlight connection succeeded`
